# Supprimer un job d'impression sur une imprimante
Pour commancer il ets nécessaire de se trouver sur le serveur d'impression.
PC Name: pge000001.dgep.edu-vaud.ch<br>
Pour se connecter au serveur d'impression il est nécessaire de passer par le serveur.
PC Name: rdfarm.dgep.edu-vaud.ch<br>
Une fois sur rdfarm ouvrez mRemoteNG
Créez une nouvelle connexion et ajouter le hostname dans le champ en en bas.
```{image} images/prisrv-mrng.png
:width: 400px
:name: prisrv-mrng
:align: center
```
Une fois dans le serveur d'impression allez dans le gestionnaire de serveur puis "tools" > "Print Management"
```{image} images/print-mng.png
:width: 600px
:name: print-mng
:align: center
```
Allez dans Printers et faites clique droit sur l'imprimante que vous souhaitez nettoyer puis "Open Printer Queue..."
```{image} images/printers-list.png
:width: 600px
:name: printers-lists
:align: center
```
Une fois dans la queue d'impression vous pouvez selecrtionner le/les jobs à supprimer.
